# 法律法规咨询助手 MVP PRD

## 1. 页面范围

MVP 包含以下页面：

- 公众咨询页
- 专业检索页
- 法规库页
- 历史记录页
- 数据源管理页
- 解析预览页
- 检索测试页

## 2. 公众咨询页

### 目标

普通用户输入法律问题，系统返回结构化、可追溯、带风险提示的回答。

### 页面结构

```text
左侧：导航
中间：对话区
右侧：引用依据 / 待确认事实 / 风险提示
```

### 核心交互

1. 用户输入法律问题。
2. 系统显示检索和生成状态。
3. Agent 返回结构化回答。
4. 回答中的依据标签可点击。
5. 右侧引用依据面板定位到对应法条。

### 回答结构

```text
一、初步判断
二、相关法律依据
三、可准备材料
四、需要确认
五、风险提示
```

### 业务规则

- 回答必须包含引用依据。
- 没有依据时不得给确定结论。
- 用户事实不足时必须列出待确认事实。
- 高风险问题必须提示咨询专业律师。
- 页面底部显示免责声明：内容仅供参考，不构成法律意见。

## 3. 专业检索页

### 目标

帮助内部专业用户快速检索、筛选、整理法律依据。

### 页面结构

```text
顶部：搜索框 + 筛选器
中间：检索结果列表
右侧：已选依据清单
```

### 筛选器

- 法规类型
- 生效状态
- 发布日期
- 法律名称

### 检索结果卡片字段

- 法规名称
- 条号
- 章节路径
- 正文摘要
- 命中方式：关键词 / 语义 / 条号
- 相关度
- 加入依据清单按钮

### 已选依据清单能力

- 移除依据
- 复制依据
- 导出 Markdown
- 生成法律依据段落

## 4. 法规库页

### 目标

浏览已入库法律法规和具体条文。

### 法规列表字段

- 法规名称
- 法规类型
- 发布机关
- 发布日期
- 生效日期
- 生效状态
- 条文数
- 索引状态

### 法规详情结构

```text
左侧：目录树，展示编/章/节/条
中间：原文条文
右侧：元数据
```

## 5. 数据源管理页

### 目标

管理法规文件导入任务。

### 功能

- 上传 docx、pdf、txt、html。
- 填写来源 URL。
- 选择法规类型。
- 填写发布机关。
- 开始解析。
- 查看解析状态。

### 列表字段

- 文件名
- 法规名称
- 来源类型
- 解析状态
- 导入时间
- 操作

### 状态

- 待解析
- 解析中
- 待确认
- 已发布
- 解析失败

## 6. 解析预览页

### 目标

确认文档是否被正确切分成条文。

### 页面结构

```text
左侧：法规目录树
中间：条文内容编辑区
右侧：解析质量检查
```

### 质量指标

- 识别条文数
- 重复条号
- 缺失条号
- 未归属段落
- 异常段落

### 操作

- 编辑条文
- 合并段落
- 拆分条文
- 确认入库
- 生成索引
- 发布知识库

### 民法典样本验收

对于 `中华人民共和国民法典_20200528.docx`，理想解析结果：

```text
法规名称：中华人民共和国民法典
识别条文数：1260
重复条号：0
缺失条号：0
状态：可发布
```

## 7. 检索测试页

### 目标

调试和评估 RAG 检索效果。

### 输入

用户输入测试问题，例如：

```text
别人欠钱不还怎么办？
```

### 输出

- 关键词检索结果
- 向量检索结果
- rerank 后结果
- 最终采用依据

### 结果字段

- 排名
- 法规名称
- 条号
- 命中方式
- 相关度
- 命中片段
- 是否采用

### 参数

- topK
- 关键词权重
- 向量权重
- 仅现行有效
- 标题命中加权

## 8. MVP 验收场景

1. 上传 `中华人民共和国民法典_20200528.docx`。
2. 解析出 1260 条。
3. 搜索 `违约责任` 能命中相关条文。
4. 搜索 `第五百七十七条` 能精确命中。
5. 问 `别人欠钱不还怎么办`。
6. 回答展示引用依据。
7. 右侧引用依据能定位到具体法条。
8. 没有检索依据时，Agent 不编造回答。
