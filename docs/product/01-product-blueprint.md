# 法律法规咨询助手 MVP 产品蓝图

## 1. 产品定位

法律法规咨询助手是一个基于权威法律法规库的法律咨询与检索工作台。

产品不是 AI 律师，不提供正式法律意见。它的核心价值是帮助用户快速找到相关法律法规，并用可追溯引用解释法律问题。

## 2. 目标用户

### 普通咨询用户

用自然语言提出法律问题，获得通俗解释、相关法律依据、可准备材料、待确认事实和风险提示。

### 内部专业用户

快速检索法律依据，整理法条，生成可复制的法律依据段落，提高律师、法务或运营人员的检索效率。

### 知识库管理员

上传法规文件，校验条文解析结果，生成索引并发布 RAG 知识库。

## 3. 核心价值

- 让普通用户看懂法律法规，而不是只看到原文条款。
- 让专业用户更快找到可引用依据。
- 让每个法律结论都有来源，降低 AI 幻觉风险。
- 让知识库可导入、可解析、可测试、可更新。

## 4. MVP 核心目标

打通最小闭环：

```text
上传法规文件
 -> 解析条文
 -> 生成知识库
 -> 用户提问
 -> 检索法条
 -> Agent 回答
 -> 展示引用依据
```

## 5. MVP 功能范围

- 公众咨询
- 专业检索
- 法规库浏览
- 数据源管理
- 解析预览
- 检索测试
- 引用依据面板
- 基础反馈评测

## 6. 非目标范围

MVP 不做以下能力：

- 胜诉率预测
- 类案裁判倾向分析
- 正式法律意见
- 复杂合同审查
- 律师入驻
- 企业合规审批
- 多租户复杂权限
- 无来源回答
- 自动爬取全网法律内容

## 7. 页面信息架构

```text
法律咨询
- 公众咨询
- 专业检索
- 法规库
- 历史记录

知识库管理
- 数据源管理
- 解析预览
- 检索测试
```

普通用户只显示法律咨询相关页面。管理员显示知识库管理页面。

## 8. RAG 知识库能力

MVP 的 RAG 知识库围绕法律法规条文建设：

- 支持上传 docx、pdf、txt、html。
- 支持按法律名称、编、章、节、条解析文档。
- 支持条文级存储。
- 支持关键词检索和向量检索。
- 支持引用回溯到具体条文。
- 支持检索测试，查看命中依据。

第一条样本数据为 `中华人民共和国民法典_20200528.docx`。

## 9. Agent 回答边界

Agent 可以做：

- 法条解释
- 法律问题初步分析
- 材料清单整理
- 待确认事实提示
- 风险提示
- 法律依据整理

Agent 不可以做：

- 承诺胜诉或败诉
- 代替律师出具正式法律意见
- 编造法律名称、条号、案例或来源
- 在没有依据时给确定结论

## 10. 成功标准

- 民法典 docx 可解析出 1260 条。
- 每条法条有条号、正文、章节路径。
- 用户问题能命中相关法条。
- Agent 回答必须带引用。
- 引用能回到原文条文。
- 没有依据时不编造。
- 公众咨询模式的回答普通用户能看懂。
- 专业检索模式能节省法条查找和依据整理时间。
